# PWA Icons

This directory should contain the following icon files for PWA support:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- apple-touch-icon.png (180x180)

For production, replace these with actual icon files that match your brand.

## Icon Requirements:

1. All icons should have transparent backgrounds
2. Use your brand colors (primary: #3b82f6)
3. Icons should be clear and recognizable at small sizes
4. Follow platform-specific guidelines for maskable icons

## Generating Icons:

You can use online tools like:
- https://realfavicongenerator.net/
- https://maskable.app/
- https://pwa-asset-generator.com/

Or use image editing software to create icons at the required sizes.
