## タスク概要
Redditパーサーが実際のMarkdownデータ形式に対応しておらず、投稿の詳細内容（本文、アップボート数、要約）が表示されない問題を修正する。

## 作業内容

### 問題の詳細
現在のパーサーが期待する形式:
```
**スコア**: 203
**投稿者**: author_name
**本文**: 投稿内容
```

実際のデータ形式:
```
本文: Kyutai（キュウタイ）は、新しいリアルタイムの...
アップボート数: 203
**要約**:
1. 投稿の主な内容...
```

### 修正要件
1. **フィールド名の修正**
   - `**スコア**:` → `アップボート数:`
   - `**本文**:` → `本文:`

2. **新しいフィールドの追加**
   - `**要約**:` セクションの抽出と表示
   - `リンク:` 情報の抽出（存在する場合）

3. **表示優先度の調整**
   - 要約が存在する場合は要約を優先表示
   - 本文は要約がない場合のフォールバック

4. **後方互換性の維持**
   - 既存の形式も引き続きサポート

### 期待される結果
- Reddit投稿の詳細情報（本文、アップボート数、要約）が正しく表示される
- data/reddit_explorer/2025-07-04.mdの豊富なデータがUIに反映される
- 各投稿で要約、アップボート数、カテゴリ別表示が適切に機能する

### テスト確認項目
- http://localhost:5173/?source=reddit で詳細内容が表示されること
- アップボート数が数値として正しく表示されること
- 要約セクションが適切にフォーマットされて表示されること
- 既存のRedditデータ（もしあれば）も引き続き表示されること
