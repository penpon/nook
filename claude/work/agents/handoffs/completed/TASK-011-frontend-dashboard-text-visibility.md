# TASK-011: LLM API使用状況ダッシュボードの文字表示問題修正

## 概要
LLM API使用状況ダッシュボードで「Usage Dashboard」と「最終更新」の文字が背景と同化して見えない問題を修正する。

## 問題詳細
- 「LLM API 使用状況ダッシュボード」のタイトル文字が背景と同化
- 「最終更新: 3:10:32」の文字が背景と同化
- 原因: MUIのテーマモードとTailwind CSSのダークモードの不整合

## 変更予定ファイル
- frontend/src/components/UsageDashboard.tsx

## 前提タスク
- なし（即座に実行可能）

## 作業内容
1. UsageDashboard.tsxのヘッダー部分の文字色設定を修正
2. MUIテーマ依存の条件分岐を削除し、確実に視認できる色に変更
3. Tailwind CSSのユーティリティクラスを使用した実装に統一

## 修正方針
- タイトル（184行目）: `color: '#1a1a1a'` に変更（ダークモード対応なし）
- 最終更新テキスト（193行目）: `color: '#4a4a4a'` に変更（ダークモード対応なし）
- または、親要素のCardコンポーネントの背景色を考慮した適切な文字色を設定

## 完了条件
