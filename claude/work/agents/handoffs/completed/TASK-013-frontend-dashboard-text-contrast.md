# TASK-013: ダッシュボードの文字色を再修正（視認性向上）

## 概要
LLM API使用状況ダッシュボードの文字が背景と同化して見えない問題を再修正する。

## 問題詳細
- 現在の文字色（#1a1a1a、#4a4a4a）がダークモードの背景に対して見えにくい
- ダークモードでもライトモードでも確実に見える色設定が必要

## 変更予定ファイル
- nook/frontend/src/components/UsageDashboard.tsx

## 前提タスク
- なし（即座に実行可能）

## 作業内容
1. タイトルの文字色を変更（176行目）
   - 現在: `sx={{ color: '#1a1a1a' }}`
   - 修正後: `sx={{ color: theme.palette.text.primary }}` または `sx={{}}` (デフォルトを使用)

2. 最終更新テキストの文字色を変更（180行目）
   - 現在: `sx={{ color: '#4a4a4a' }}`
   - 修正後: `sx={{ color: theme.palette.text.secondary }}` または `sx={{}}` (デフォルトを使用)

3. MUIのデフォルトテーマ色を使用することで、自動的にライト/ダークモードに対応

## 完了条件
- ダークモードでもライトモードでも文字が明確に読める
- MUIのテーマシステムに準拠した実装
